Prompt 1: Entertainment news article scraping
In scraper.py, write a function called scrape_entertainment(page) that:
- Navigates to https://ekantipur.com/entertainment using Playwright
- Waits for the page to load
- Extracts top 5 articles, each with: title, image_url, category, author
- Returns a list of dictionaries
- Handles errors with try/except and returns empty list on failure

Prompt 2: Cartoon of the day scraping
In scraper.py, write a separate function called scrape_cartoon(page) that:
- Navigates to https://ekantipur.com/cartoon using Playwright
- Waits for the page to load
- Extracts the cartoon of the day with: title, image_url, author
- Returns a single dictionary
- Handles errors with try/except and returns None values on failure

Prompt 3: Helper functions
Write two helper functions for Playwright scraping:
1. safe_text(element, selector) — returns inner text of child element or None if not found
2. safe_attr(element, selector, attr) — returns attribute value of child element or None
Wrap both in try/except so they never crash the script.

Prompt 6: main function
In scraper.py, write a main() function that:
- Launches Chromium browser using Playwright with headless=False
- Creates a browser context with a real Chrome user_agent, viewport 1280x800, locale ne-NP
- Opens a new page
- Calls scrape_entertainment(page) and scrape_cartoon(page)
- Stores results in a dictionary with keys entertainment_news and cartoon_of_the_day
- Closes the browser after scraping
